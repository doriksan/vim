## .vimrc
```
set number "нумерация строк
set expandtab "заменяем табы на пробелы
"set tabstop=2 "размер таба - 2 пробела
   
set hlsearch "подсветка поисковых резкльтатов
set incsearch "инкрементальный поиск
```
